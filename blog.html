Question-1: What are the difference ways to select an elements in the DOM?

To select elements in the DOM there are mainly 4 ways. Like getElementById to select by ID, getElementsByClassName to select by class name, getElementsByTagName to select by tag name, and CSS Selectors like querySelector and querySelectorAll.

Question-2: What is the difference between innerHTML,innerText,and textContent ?

innerHTML gets or sets the HTML markup contained within an element. This includes both text and HTML tags.
innerText gets or sets the visible text content of an element.
textContent gets or sets the text content of an element, It also returns all text, whether it's hidden on screen or not. ,

Question-3: What is event Deligation in the DOM?

Event Delegation is a technique in JavaScript where attach a single event handler to a parent element to listen for events that happen on its child elements. Instead of adding event listeners to each individual child element, delegate the event handling to the parent.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is the process where when an event occurs on an element, it triggers not only on that element but also propagates upwards through its parent elements in the DOM hierarchy, essentially bubbling up until it reaches the root element of the document.
